# Базовый промпт
Я пишу платформу для создания разнообразных тестов специально для учителей английского языка с возможностью последующего тестирования учеников.
Технологический стек - монорепозиторий с фронтендом: Next.js + Tailwind; бэкендом: Nest.js + TypeORM + PostgreSQL. 
Ее первая итерация - модуль для прохождения словарного диктанта, где по картинкам ученики должны вписывать в точности подходящее слово на английском. Их результат отображается после прохождения - количество баллов и красными ошибки = отличия от заданного слова.
