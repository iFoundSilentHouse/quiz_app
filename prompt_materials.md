# Базовый промпт
Я пишу платформу для создания разнообразных тестов специально для учителей английского языка с возможностью последующего тестирования учеников.
Технологический стек - монорепозиторий с фронтендом: Next.js + Tailwind; бэкендом: Nest.js + TypeORM + PostgreSQL. 
Ее первая итерация - модуль для прохождения словарного диктанта, где по картинкам ученики должны вписывать в точности подходящее слово на английском. Их результат отображается после прохождения - количество баллов и красными ошибки = отличия от заданного слова.

Шаг 1 — Общие типы в shared
Определить TypeScript-интерфейсы, которые будут использоваться и на фронте, и на бэке:

Question — вопрос (id, imageUrl, correctAnswer)
QuizAttempt — попытка прохождения (id, quizId, studentName, answers, score, createdAt)
AnswerResult — результат одного ответа (questionId, given, correct, diff)
Шаг 2 — База данных и ORM (Backend)
Настройка:

Поднять PostgreSQL (локально через Docker Compose)
Подключить TypeORM к Nest.js, настроить DataSource
Настроить миграции
Сущности (entities):
Сущность	Поля
Quiz	id, title, description, createdAt
Question	id, imageUrl, correctAnswer, order, quiz (FK)
QuizAttempt	id, studentName, score, quiz (FK), createdAt
AttemptAnswer	id, givenAnswer, isCorrect, attempt (FK), question (FK)
Шаг 3 — Backend API (Nest.js)
Модуль quiz (логика учителя):

POST /quizzes — создать тест
POST /quizzes/:id/questions — добавить вопрос с картинкой
GET /quizzes — список тестов
GET /quizzes/:id — тест с вопросами
PUT /quizzes/:id — редактировать тест
DELETE /quizzes/:id — удалить тест
Модуль attempt (логика ученика):

POST /quizzes/:id/attempts — сохранить попытку, посчитать score, записать diff каждого ответа
GET /attempts/:id — получить результаты попытки
Сервис проверки ответов (AnswerCheckerService):

Сравнение строк (без учёта регистра, с trim)
Вычисление посимвольного diff для подсветки ошибок
